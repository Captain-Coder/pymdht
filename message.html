message¶

The message module contains all the data structures needed to create, encode, and decode valid MDHT messages.

Outgoing messages are built from a few parameters. They are immutable and can oly be stamped once.

Incoming messages are built from bencoded data. They are automatically sanitized.

class message.IncomingMsg(datagram)[source]¶

Create an object by decoding the given Datagram object. Raise ‘MsgError’ whenever the decoder fails to decode the datagram’s data (e.g., invalid bencode).

stamp(tid)[source]¶

Return a Datagram object ready to be sent over the network. The message’s state is changed internally to reflect that this message has been stamped. This call will raise MsgError if the message has already been stamped.
